SWAP MEMORY
Probaly you may solve this problem by incresing your swap memory.
Fisrt turn off your swap memory

	$ sudo dphus-swapfile swapoff


Go to swapfile and change the amount of swap-memory

	$  sudo nano /etc/dphus-swapfile


The default config swapsize is 100(1Mi), you can change that to 2048 (2Gi)

	# CONF_SWAPSIZE=100
	CONF_SWAPSIZE=2048

After you can setup the memory, and turn on the swapmemory

	$ sudo dphus-swapfile setup
	$ sudo dphus-swapfile swapon


Upgrade OS:
	$ sudo apt update && sudo apt upgrade -y
	

Connect the Rasp in PC:

	$ ifconfig
	# ** see the ip

	$ sudo poweroff

	# In the PC:
		$ ssh pi@<ip>


Swap memory:

	$ sudo dphys-swapfile swapoff
	$ sudo nano /etc/dphys-swapfile

	# ** Set:
		CONF_SWAPSIZE=2048
	
	$ sudo dphus-swapfile setup
	$ sudo dphus-swapfile swapon


Active the X11 tunneling:
	$ sudo nano /ect/shh/sshd_config

	# ** Set:
		X11Forwarding yes
		X11DisplayOffset 10
		X11UseLocalhost yes

Configure the serial port:

	# ** Check if the serial port activated:
	$ ls /dev/tty*
	
	# ** Speed up the UART to enable fast connection with the FCU
	$ sudo nano /boot/config.txt
	
	# ** Set:
		enable_uart=1
		init_uart_clock=64000000

	$sudo reboot
